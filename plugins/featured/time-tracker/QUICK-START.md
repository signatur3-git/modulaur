## â“ FAQ

**Q: Can I edit a time entry after saving?**  
A: Yes! Click the âœï¸ icon on any entry to edit it.

**Q: What if I forget to log time?**  
A: Just create a new entry with the correct past times. The system accepts any date/time.

**Q: Can I delete a project?**  
A: Yes, but time entries will remain. They just won't link to a project anymore.

**Q: How do I track non-billable time?**  
A: Uncheck the "Billable" checkbox when creating the entry.

**Q: Can I use this on multiple computers?**  
A: Currently, data is stored locally. You'd need to export/import or use the same computer.

**Q: What's the difference between Time Tracker page and panels?**  
A: The page gives you a complete workspace with tabs. Panels are individual widgets you can add to any dashboard.

**Q: Can I add the Timer Panel to this page?**  
A: The page focuses on manual entry. But you can add TimerPanel to a custom dashboard if you prefer timers.
